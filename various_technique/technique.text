다익스트라의 경우 음수 간선을 허용하지 않는다
플로이드의 경우 음수 간선을 허용하나 음수의 사이클이 있는 그래프에선 허용하지 않는다
음수 사이클의 경우 벨만 포드 알고리즘을 사용해야 한다

2차원 배열의 경우 값만 복사하려면
import copy
copy.deepcopy(2차원 배열)
의 구성으로 해야한다

DFS 를 비롯한 제귀 메소드는 divide and conquer 의 알고리즘이 유용할 수 있다
단, dp 를 구성할 때 divide and conquer 가 같은 값을 여러번 부른다면
dp를 직접 구성하는 것이 더 빠를 수 있다
